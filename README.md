# 수행평가·세특 활동 탐색

정적 HTML/CSS/JavaScript로 만든 활동 검색·비교 도구입니다.

## 현재 목적

학생이 개인정보 없이 수행평가·세특 활동 아이디어를 검색하고,
조건에 맞는 활동을 고르고,
여러 활동을 비교해 볼 수 있도록 돕는 도구입니다.

## 현재 파일 구성

- `index.html`
- `styles.css`
- `app.js`
- `activities.js`

## 현재 구현된 기능

- 키워드 검색
- 다중 필터 검색
- 검색 결과 개수 표시
- 활동 카드 목록 표시
- 활동 상세 보기
- 개별 활동 복사
- 활동 계획서 초안 복사
- 활동 담기
- 담은 활동 일괄 복사
- 담은 활동 2개 비교
- 비교표 내 선택 투표
- 모바일 대응 UI

## 현재 비교 기능 동작

- 담은 활동 목록에서 바로 `비교 선택` 체크
- 2개를 선택하면 비교표 자동 표시
- 비교표에서는 핵심 항목 먼저 표시
- `자세한 항목 보기`를 누르면 추가 항목 표시
- 각 비교 칸을 누르면 더 마음에 드는 쪽으로 점수 표시

## 현재 데이터 해석 원칙

- `관련 개념 키워드`는 엄밀한 교과 단원명이 아니라 탐구와 연결되는 개념 키워드입니다.
- 이후 데이터 확장 시, 실제 단원명 또는 성취기준 수준의 정보는 별도 필드로 추가하는 것이 좋습니다.

## 다음에 다듬으면 좋은 것

1. 활동 데이터 20~30개 이상 추가
2. `관련 개념 키워드`를 과목 단원 수준으로 더 정교화할지 검토
3. 비교표 핵심 항목 순서 재점검
4. 활동 방식/수행평가 형식 태그 정렬 규칙 통일
5. 활동 데이터 표현 통일

활동1

2-2: 빅데이터 시각화 / 학급 내 생활 데이터(수면·식습관·성적 등) 직접 수집해 엑셀·구글 시트로 시각화 분석 / ★수집 변인·시각화 방식 변경 가능
3-1: 데이터 보안 탐구보고서 / 암호화 방식별 보안 강도·처리 속도 비교 실험 + 개인정보 보호 인식 설문 수집 / ★암호화 방식·설문 대상 변경 가능

1-2: AI 윤리 인식 설문 / AI 활용 경험·윤리적 우려·신뢰도를 학급·교사·학부모 대상 설문 수집 비교 / ★설문 대상·문항 변경 가능
2-1: 이미지 압축 품질 비교 / 같은 이미지를 다양한 압축률로 저장해 용량·화질·파일 크기 변화 직접 측정 비교 / ★이미지 종류·알고리즘 변경 가능
2-2: AI 추천 알고리즘 분석 / 유튜브·넷플릭스·스포티파이의 추천 패턴을 직접 관찰·기록해 알고리즘 작동 방식 비교 분석 / ★플랫폼 종류·관찰 변인 변경 가능

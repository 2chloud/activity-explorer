계열별 탐구활동 로드맵 (5단계)
형식: 활동명 / 예시 / ★(변형 아이디어)
단계: 1학년1학기 → 1학년2학기 → 2학년1학기 → 2학년2학기 → 3학년1학기
계열: 농수축산

농수축산 최종 정리본
활동1
1학년 1학기: 토양 종류와 작물 성장 비교 실험
예시: 모래·점토·부엽토·혼합토에서 같은 씨앗을 발아시켜 생장 속도·뿌리 길이·잎 크기를 직접 측정 비교
★ 토양 종류·작물 유형을 바꿔서 설계해도 좋아요
1학년 2학기: 식물 생장 호르몬 효과 실험
예시: 옥신·지베렐린 처리 농도별 콩나물·무순의 생장 속도·뿌리 길이를 직접 측정 비교
★ 호르몬 종류·농도·식물 유형을 바꿔서 설계해도 좋아요
2학년 1학기: 환경 조건별 작물 성장 비교 실험
예시: 온도·빛·수분 조건을 달리해 같은 작물의 생장 속도·수확량을 직접 측정 비교
★ 환경 변인·작물 종류를 바꿔서 설계해도 좋아요
2학년 2학기: 배양육·대체 단백질 인식 설문 조사
예시: 배양육·곤충·식물성 단백질에 대한 학급·교사·학부모 인식·수용도를 직접 설문 수집해 세대별 비교
★ 대체 단백질 유형·설문 대상을 바꿔서 설계해도 좋아요
3학년 1학기: 축산업 환경 영향 탐구보고서
예시: 축산업 환경 영향(탄소·토지·수질)에 대한 학급·학부모·농업 종사자 인식을 직접 설문 수집 후 탐구보고서 작성
★ 환경 영향 항목·설문 대상을 바꿔서 설계해도 좋아요
활동2
1학년 1학기: 농업과 기후 변화 인식 설문 조사
예시: 기후 변화가 농업에 미치는 영향에 대한 학급·교사·학부모·농업 종사자 인식을 직접 설문 수집해 비교 분석
★ 기후 변화 항목·설문 대상을 바꿔서 설계해도 좋아요
1학년 2학기: GMO·자연 교배 작물 비교 실험
예시: GMO·비GMO 식품의 성분표·외관·맛을 직접 비교하고 학급 내 인식·선호도 블라인드 테스트 수집
★ 식품 종류·비교 항목을 바꿔서 설계해도 좋아요
2학년 1학기: 폐기물 활용 친환경 농업 실험
예시: 커피 찌꺼기·달걀껍데기·음식물 퇴비를 토양에 혼합해 작물 생장 속도를 직접 측정 비교
★ 폐기물 종류·혼합 비율을 바꿔서 설계해도 좋아요
2학년 2학기: 대체 단백질 소비 변화 설문 조사
예시: 육류 소비 줄이기·채식·대체 단백질 선택 경험을 학급·교사·학부모 대상 직접 설문 수집해 세대별 비교
★ 단백질 유형·설문 대상을 바꿔서 설계해도 좋아요
3학년 1학기: 기후 변화 대응 농업 탐구보고서
예시: 스마트팜·수직농업·도시농업 현장을 직접 방문 또는 사진·데이터 수집하고 농업 종사자 인터뷰 후 탐구보고서 작성
★ 농업 기술 유형·조사 지역을 바꿔서 설계해도 좋아요
활동3
1학년 1학기: 곤충과 농업 생태계 관찰 실험
예시: 교내 화단·텃밭에서 곤충 종류·개체 수를 시기별로 직접 관찰·기록하고 작물 생장과의 관계 분석
★ 관찰 장소·시기를 바꿔서 설계해도 좋아요
1학년 2학기: 도시농업·수직농업 효율 비교 실험
예시: 교내 화분·수직 재배기·일반 토양에서 같은 작물을 재배해 생장 속도·수확량을 직접 측정 비교
★ 재배 방식·작물 종류를 바꿔서 설계해도 좋아요
2학년 1학기: 수온 변화와 수생 생물 영향 실험
예시: 어항 수온을 달리해 금붕어·열대어의 활동량·먹이 섭취량 변화를 직접 관찰·기록 비교
★ 수온 범위·수생 생물 종류를 바꿔서 설계해도 좋아요
2학년 2학기: 가축 사료와 성장 비교 실험
예시: 다양한 먹이(일반 사료·유기농·채소) 조건별 곤충(밀웜·귀뚜라미) 성장 속도를 직접 측정 비교
★ 먹이 종류·곤충 유형을 바꿔서 설계해도 좋아요
3학년 1학기: 대체 단백질 경제성 탐구보고서
예시: 배양육·식물성 고기·곤충 단백질의 가격·영양·환경 비용을 직접 수집 비교 + 소비자 인식 설문 후 탐구보고서 작성
★ 대체 단백질 유형·비교 항목을 바꿔서 설계해도 좋아요
활동4
1학년 1학기: 농업과 생물 다양성 현장 조사
예시: 학교 주변 농지·화단·텃밭의 생물 종류·개체 수를 직접 관찰·기록하고 농약 사용 여부와 비교 분석
★ 관찰 장소·생물 유형을 바꿔서 설계해도 좋아요
1학년 2학기: 곤충 단백질 인식 설문 조사
예시: 곤충 단백질 식품(귀뚜라미 과자·밀웜 단백질바 등) 시식 후 학급 내 반응·수용도를 직접 측정 비교
★ 곤충 단백질 종류·측정 항목을 바꿔서 설계해도 좋아요
2학년 1학기: 친환경 수산물 인식 설문 조사
예시: 친환경·양식·자연산 수산물에 대한 학급·교사·학부모 인식·구매 의향을 직접 설문 수집해 비교 분석
★ 수산물 종류·설문 대상을 바꿔서 설계해도 좋아요
2학년 2학기: 가축 전염병 예방 인식 설문 조사
예시: 조류독감·아프리카돼지열병 등 가축 전염병에 대한 학급·학부모·농업 종사자 인식을 직접 설문 수집 비교
★ 전염병 종류·설문 대상을 바꿔서 설계해도 좋아요
3학년 1학기: 유전자 편집 농산물 인식 탐구보고서
예시: CRISPR 기술로 개발된 유전자 편집 농산물에 대한 학급·교사·학부모 인식을 직접 설문 수집 분석 후 탐구보고서 작성
★ 농산물 종류·설문 대상을 바꿔서 설계해도 좋아요

계열별 탐구활동 로드맵 (5단계)
형식: 활동명 / 예시 / ★(변형 아이디어)
단계: 1학년1학기 → 1학년2학기 → 2학년1학기 → 2학년2학기 → 3학년1학기
계열: 교육

교육 최종 정리본
활동1
1학년 1학기: 교사 역할 인식 세대별 비교 조사
예시: 학급·교사·학부모·조부모 대상 이상적인 교사상을 직접 설문·인터뷰로 수집해 세대별 비교 분석
★ 설문 대상·교사 역할 항목을 바꿔서 설계해도 좋아요
1학년 2학기: 학생 중심·교사 중심 수업 효과 비교 실험
예시: 같은 내용을 강의식·토론식으로 각각 수업 후 학급 내 이해도·만족도를 사전·사후 설문으로 직접 비교 측정
★ 수업 방식·측정 변인을 바꿔서 설계해도 좋아요
2학년 1학기: 맞춤형 교수법 효과 비교 실험
예시: 같은 내용을 시각·청각·활동 중심으로 달리 제시해 학급 내 이해도·기억력을 직접 측정 비교
★ 교수법 유형·측정 방법을 바꿔서 설계해도 좋아요
2학년 2학기: 보상·처벌이 학습 태도에 미치는 영향 실험
예시: 칭찬·피드백·무반응 조건별 학습 과제 수행 결과를 학급 내 직접 실험으로 수집 비교
★ 보상 유형·과제 종류를 바꿔서 설계해도 좋아요
3학년 1학기: 문해력 교육 효과 탐구보고서
예시: 독서·요약·토론 방식별 문해력 변화를 사전·사후 테스트로 직접 측정 비교 후 탐구보고서 작성
★ 교육 방식·측정 문항을 바꿔서 설계해도 좋아요
활동2
1학년 1학기: 유아·초등 학습 방식 인식 비교 조사
예시: 유아·초등 교육 경험자(학급·학부모·교사) 대상 학습 방식 차이·기억을 직접 인터뷰·설문으로 수집 비교
★ 교육 단계·설문 항목을 바꿔서 설계해도 좋아요
1학년 2학기: 교육·민주주의 인식 설문 조사
예시: 학급·교사·학부모 대상 학교 내 민주적 의사결정 경험·인식을 직접 설문 수집해 세대별 비교
★ 설문 대상·민주주의 항목을 바꿔서 설계해도 좋아요
2학년 1학기: 디지털·종이 교과서 학습 효과 비교 실험
예시: 같은 내용을 디지털·종이 교과서로 각각 학습 후 이해도·집중도를 사전·사후 테스트로 직접 비교 측정
★ 교과목·측정 변인을 바꿔서 설계해도 좋아요
2학년 2학기: 학습 환경과 집중도 측정 실험
예시: 소음·조명·온도 조건을 달리해 학급 내 집중도·과제 수행 시간을 직접 측정 비교
★ 환경 변인·측정 방법을 바꿔서 설계해도 좋아요
3학년 1학기: 다문화 교육 인식 탐구보고서
예시: 다문화 가정 학생·학부모 대상 학교 적응 경험·어려움을 직접 인터뷰·설문 수집 후 탐구보고서 작성
★ 대상 집단·인터뷰 항목을 바꿔서 설계해도 좋아요
활동3
1학년 1학기: 동서양 교육 철학 인식 비교 조사
예시: 학급·교사·학부모 대상 교육의 목적·방식에 대한 인식을 직접 설문 수집해 세대별 비교 분석
★ 설문 대상·교육 철학 항목을 바꿔서 설계해도 좋아요
1학년 2학기: 교사 질문법과 학생 사고력 비교 실험
예시: 단답형·개방형·소크라테스식 질문을 각각 제시해 학급 내 사고 깊이·응답 다양성을 직접 비교 측정
★ 질문 유형·측정 방법을 바꿔서 설계해도 좋아요
2학년 1학기: 게임 기반 학습 효과 비교 실험
예시: 같은 내용을 게임·강의·퀴즈 방식으로 각각 학습 후 이해도·흥미도를 사전·사후로 직접 비교 측정
★ 게임 유형·교과 내용을 바꿔서 설계해도 좋아요
2학년 2학기: 교사 피드백 유형별 학습 효과 실험
예시: 즉각·지연·긍정·교정 피드백을 각각 제공해 학급 내 과제 수행 결과 변화를 직접 비교 측정
★ 피드백 유형·과제 종류를 바꿔서 설계해도 좋아요
3학년 1학기: 교육 격차 실태 탐구보고서
예시: 학급 내 학습 환경·사교육 경험 차이를 직접 설문 수집 분석하고 교육 격차 인식 비교 후 탐구보고서 작성
★ 격차 유형·설문 대상을 바꿔서 설계해도 좋아요
활동4
1학년 1학기: 교육과 경제 인식 비교 조사
예시: 학급·교사·학부모 대상 교육이 경제적 성공에 미치는 영향 인식을 직접 설문 수집해 세대별 비교
★ 설문 대상·경제 항목을 바꿔서 설계해도 좋아요
1학년 2학기: 토론식·강의식 수업 효과 비교 실험
예시: 같은 내용을 토론식·강의식으로 각각 수업 후 이해도·참여도·기억력을 사전·사후로 직접 비교 측정
★ 수업 방식·교과 내용을 바꿔서 설계해도 좋아요
2학년 1학기: 조별·개별 학습 효과 비교 실험
예시: 같은 과제를 조별·개별로 각각 수행해 결과물 완성도·소요 시간·만족도를 직접 비교 측정
★ 과제 유형·집단 크기를 바꿔서 설계해도 좋아요
2학년 2학기: 예방 교육 효과 비교 실험
예시: 강의·영상·역할극 방식의 예방 교육(안전·성·약물 등) 효과를 사전·사후 인식 설문으로 직접 비교 측정
★ 예방 교육 종류·방식을 바꿔서 설계해도 좋아요
3학년 1학기: 특수 교육 인식 탐구보고서
예시: 특수 교육 대상 학생·교사·학부모 대상 특수 교육 경험·필요성 인식을 직접 설문·인터뷰 수집 후 탐구보고서 작성
★ 특수 교육 유형·설문 대상을 바꿔서 설계해도 좋아요

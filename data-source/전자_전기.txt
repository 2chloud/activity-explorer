계열별 탐구활동 로드맵 (5단계)
형식: 활동명 / 예시 / ★(변형 아이디어)
단계: 1학년1학기 → 1학년2학기 → 2학년1학기 → 2학년2학기 → 3학년1학기
계열: 전자/전기

⚡ 전자/전기
활동1
1-1: 가전제품 전력 소비량 비교 / 스마트폰 충전기·형광등·LED·선풍기 등의 전력 소비량 전력계로 직접 측정 / ★기기 종류·사용 시간 변경 가능
1-2: 송전 손실 비교 / 도선 길이·굵기·재질별 저항값 멀티미터로 직접 측정해 송전 손실과 연결 / ★도선 재질 변경 가능
2-1: 간이 논리 회로 / LED·스위치·건전지로 AND·OR·NOT 논리 회로 직접 제작해 동작 원리 확인 / ★회로 조합·LED 색 변경 가능
2-2: 안테나 수신 효율 비교 / 안테나 길이·형태별 라디오 수신 감도 직접 비교 측정 / ★안테나 재질·방향 변경 가능
3-1: 스마트 그리드 탐구보고서 / 태양광·건전지·수동 발전 소형 모델을 조합해 전력 분배 효율 비교 / ★전력 저장 방식·부하 변경 가능
활동2
1-1: 태양광 패널 효율 측정 / 빛의 세기·각도·색에 따른 소형 태양광 패널 발전량 전류계로 직접 측정 / ★패널 크기·거리 변경 가능
1-2: 무선 충전 원리 / 코일 감는 수·간격·재질별 무선 전력 전송 효율 직접 측정 비교 / ★코일 크기·전류 세기 변경 가능
2-1: 축전기 충전·방전 / 커패시터 용량별 충전·방전 시간 직접 측정해 그래프로 비교 / ★저항값·전압 변경 가능
2-2: 광섬유 빛 전달 / 플라스틱 광섬유·투명 아크릴봉의 빛 전달 효율과 손실 직접 측정 비교 / ★광섬유 길이·굴절 각도 변경 가능
3-1: 태양전지 원리 탐구보고서 / 빛의 파장·세기·온도 조건별 소형 태양전지 발전 효율 비교 / ★태양전지 종류·표면 재질 변경 가능
활동3
1-1: 정전기와 도체·부도체 / 풍선·빗·플라스틱 막대로 정전기 발생시켜 재료별 대전량 직접 비교 / ★재료 종류·마찰 횟수 변경 가능
1-2: 발전 방식별 효율 비교 / 소형 수력·풍력·태양광 발전 모델의 발전량 직접 측정 비교 / ★발전 조건 변경 가능
2-1: 디지털·아날로그 신호 비교 / 아날로그·디지털 신호의 잡음 저항성을 간이 회로로 직접 비교 측정 / ★신호 주파수·전압 변경 가능
2-2: 위성 통신 원리 / 전파 반사체 재질·각도별 신호 반사 효율을 간이 장치로 직접 측정 비교 / ★반사체 크기·재질 변경 가능
3-1: 차세대 배터리 탐구보고서 / 리튬이온·니켈수소·알칼리 건전지의 방전 속도·용량·온도 특성 직접 측정 비교 / ★부하 저항·온도 조건 변경 가능
활동4
1-1: 전자기 유도 / 코일 감는 수·자석 속도·자석 세기 달리해 유도 전류 크기 직접 측정 비교 / ★코일 재질·자석 크기 변경 가능
1-2: 전기자동차 회생제동 원리 / 소형 모터를 발전기로 활용해 제동 시 발생하는 전력량 직접 측정 비교 / ★모터 크기·제동 속도 변경 가능
2-1: 로봇 센서 비교 / 초음파·적외선 센서의 거리 측정 정확도와 오차 직접 측정 비교 / ★측정 거리·장애물 재질 변경 가능
2-2: 자율주행 장애물 감지 / 초음파·적외선 센서 조합으로 간이 장애물 감지 회로 직접 제작 후 감지 정확도 비교 / ★센서 배치·감지 거리 변경 가능
3-1: 우주 태양광 발전 탐구보고서 / 태양광 패널 각도·거리·차폐 조건별 발전 효율 비교 측정 + 우주 환경 적용 가능성 탐구 / ★패널 종류·광원 조건 변경 가능

계열별 탐구활동 로드맵 (5단계)
형식: 활동명 / 예시 / ★(변형 아이디어)
단계: 1학년1학기 → 1학년2학기 → 2학년1학기 → 2학년2학기 → 3학년1학기
계열: 영어/일본어/중국어

영어/일본어/중국어 최종 정리본
활동1 (영어)
1학년 1학기: 영어권 국가별 발음·표현 차이 조사
예시: 미국·영국·호주·캐나다 원어민 또는 영어 교사 대상 국가별 발음·표현 차이를 직접 인터뷰·녹음 수집 비교
★ 비교 국가·언어 항목을 바꿔서 설계해도 좋아요
1학년 2학기: 뉴스 문체 독자 반응 비교 실험
예시: 같은 사건을 다룬 한국·영미 뉴스 기사를 직접 수집해 문체·표현 차이를 분석하고 학급 내 이해도 비교 측정
★ 뉴스 종류·비교 문항을 바꿔서 설계해도 좋아요
2학년 1학기: 영어 동사 사용 패턴 조사
예시: 영어 원어민·한국인 영어 학습자의 동사 사용 패턴 차이를 직접 인터뷰·텍스트 수집으로 비교 분석
★ 동사 유형·대상 그룹을 바꿔서 설계해도 좋아요
2학년 2학기: 영어 계층·지역별 언어 차이 조사
예시: 영국·미국의 계층·지역별 언어 차이 사례를 직접 수집하고 학급 내 인식 설문 비교
★ 비교 지역·언어 차이 유형을 바꿔서 설계해도 좋아요
3학년 1학기: 미래 영어 교육 탐구보고서
예시: AI·앱·전통 방식의 영어 학습 효과를 학급 내 직접 실험으로 비교 측정 후 탐구보고서 작성
★ 학습 방식·측정 변인을 바꿔서 설계해도 좋아요
활동2 (영어 심화)
1학년 1학기: 한국어·영어 시제 인식 비교 조사
예시: 같은 상황을 한국어·영어로 표현할 때 시제 선택 차이를 학급 내 직접 실험으로 수집 비교
★ 상황 유형·비교 언어를 바꿔서 설계해도 좋아요
1학년 2학기: 영어 문학 속 사회적 가치 독자 반응 조사
예시: 영어 문학 작품 속 사회적 가치(인종·성별·계층 등)에 대한 학급 내 인식을 직접 설문 수집 비교
★ 작품 종류·사회적 가치 항목을 바꿔서 설계해도 좋아요
2학년 1학기: 영어 발음 오류 패턴 수집 실험
예시: 한국인 영어 학습자의 발음 오류를 직접 녹음·수집해 유형별 분류 후 원어민 반응 비교
★ 발음 항목·학습자 수준을 바꿔서 설계해도 좋아요
2학년 2학기: 영어 프레젠테이션 효과 비교 실험
예시: 같은 내용을 다른 방식(데이터 중심·스토리 중심·시각 중심)으로 발표해 학급 내 설득력·이해도 비교 측정
★ 발표 방식·평가 기준을 바꿔서 설계해도 좋아요
3학년 1학기: 영어 학습자 오류 패턴 탐구보고서
예시: 학급 내 영어 작문·말하기 오류를 직접 수집 분류하고 원인 분석 후 탐구보고서 작성
★ 오류 유형·수집 방법을 바꿔서 설계해도 좋아요
활동3 (일본어)
1학년 1학기: 한·일 문화 공통점·차이 비교 조사
예시: 한국·일본 문화(음식·예절·학교생활 등) 공통점·차이를 학급 내 직접 설문·인터뷰로 수집 비교
★ 문화 항목·비교 방식을 바꿔서 설계해도 좋아요
1학년 2학기: 애니메이션 속 언어와 실제 일본어 비교 조사
예시: 일본 애니메이션 대화 표현을 직접 수집해 실제 일본어 화자의 사용 여부를 인터뷰로 비교 확인
★ 애니메이션 종류·표현 유형을 바꿔서 설계해도 좋아요
2학년 1학기: 일본어 신조어 사용 실태 조사
예시: 일본어 신조어·인터넷 언어를 직접 수집해 세대별 인지도·사용 빈도를 설문으로 비교 분석
★ 신조어 유형·설문 대상을 바꿔서 설계해도 좋아요
2학년 2학기: 한·일 존댓말 체계 비교 실험
예시: 같은 상황에서 한국어·일본어 존댓말 사용 방식 차이를 직접 수집해 모국어 화자 반응 비교
★ 상황 유형·비교 항목을 바꿔서 설계해도 좋아요
3학년 1학기: 일본어 생략 표현 탐구보고서
예시: 일본어 문장에서 생략되는 주어·목적어 패턴을 직접 수집 분석하고 한국어와 비교 후 탐구보고서 작성
★ 생략 유형·수집 텍스트를 바꿔서 설계해도 좋아요
활동4 (중국어)
1학년 1학기: 한·중 문화 공통점·차이 비교 조사
예시: 한국·중국 문화(음식·명절·예절 등) 공통점·차이를 학급 내 직접 설문·인터뷰로 수집 비교
★ 문화 항목·비교 방식을 바꿔서 설계해도 좋아요
1학년 2학기: 한국어·중국어 어순 비교 실험
예시: 같은 문장을 한국어·중국어·영어로 표현할 때 어순 차이를 직접 수집해 모국어 화자 반응 비교
★ 문장 유형·비교 언어를 바꿔서 설계해도 좋아요
2학년 1학기: 중국어 인터넷 신조어 사용 실태 조사
예시: 중국어 인터넷 신조어를 직접 수집해 세대별·지역별 인지도·사용 빈도를 설문으로 비교 분석
★ 신조어 유형·설문 대상을 바꿔서 설계해도 좋아요
2학년 2학기: 한자·중국어 역사 인식 비교 조사
예시: 한국·중국·일본의 한자 사용 방식 차이를 직접 수집하고 학급 내 한자 인식·학습 경험 설문 비교
★ 한자 유형·비교 국가를 바꿔서 설계해도 좋아요
3학년 1학기: 디지털 중국어 글쓰기 탐구보고서
예시: SNS·메신저·공식 문서의 중국어 표현 방식 차이를 직접 수집 분석하고 사용자 인식 설문 수집 후 탐구보고서 작성
★ 글쓰기 유형·수집 텍스트를 바꿔서 설계해도 좋아요

계열별 탐구활동 로드맵 (5단계)
형식: 활동명 / 예시 / ★(변형 아이디어)
단계: 1학년1학기 → 1학년2학기 → 2학년1학기 → 2학년2학기 → 3학년1학기
계열: 식품

식품 최종 정리본
활동1
1학년 1학기: 가공·신선 식품 영양소 비교 실험
예시: 같은 종류의 가공식품·신선식품(과일주스 vs 생과일 등)의 당·나트륨·비타민 함량을 성분표로 직접 수집 비교
★ 식품 종류·영양소 항목을 바꿔서 설계해도 좋아요
1학년 2학기: 김치 발효 과정 pH 변화 실험
예시: 직접 담근 김치의 발효 일수별 pH 변화를 pH 시험지·pH미터로 직접 측정 기록
★ 김치 재료·보관 온도를 바꿔서 설계해도 좋아요
2학년 1학기: 식품 가공 과정 영양소 변화 실험
예시: 같은 식재료를 생·삶기·볶기·튀기기 방식으로 조리해 색·질감·비타민C 함량 변화를 직접 측정 비교
★ 식재료 종류·조리 방법을 바꿔서 설계해도 좋아요
2학년 2학기: 식품 첨가물 인식 설문 조사
예시: 햄·소시지·과자 등 가공식품 첨가물에 대한 학급·교사·학부모 인식을 직접 설문 수집해 세대별 비교
★ 첨가물 종류·설문 대상을 바꿔서 설계해도 좋아요
3학년 1학기: 인공·천연 감미료 인식 탐구보고서
예시: 설탕·스테비아·아스파탐의 단맛 인식 역치를 학급 내 블라인드 테스트로 직접 측정 + 인식 설문 수집 후 탐구보고서 작성
★ 감미료 종류·피험자 그룹을 바꿔서 설계해도 좋아요
활동2
1학년 1학기: 식품 방부제 효과 비교 실험
예시: 소금·식초·설탕·꿀 농도별 식품 보존 효과를 빵·과일 변화로 직접 관찰·기록 비교
★ 방부제 종류·농도를 바꿔서 설계해도 좋아요
1학년 2학기: 콩 발효 과정 비교 실험
예시: 된장·청국장·간장의 발효 기간별 색·냄새·pH 변화를 직접 측정·기록 비교
★ 발효 식품 종류·측정 변인을 바꿔서 설계해도 좋아요
2학년 1학기: 대체 감미료 단맛 비교 실험
예시: 설탕·스테비아·자일리톨·올리고당의 단맛 강도를 학급 내 블라인드 테스트로 직접 측정 비교
★ 감미료 종류·농도를 바꿔서 설계해도 좋아요
2학년 2학기: 식물성·동물성 단백질 비교 실험
예시: 두부·콩·닭가슴살·달걀의 단백질 함량·조리 후 질감 변화를 직접 측정·비교하고 학급 내 선호도 설문 수집
★ 단백질 식품 종류·측정 변인을 바꿔서 설계해도 좋아요
3학년 1학기: 유전자 변형 식품 인식 탐구보고서
예시: GMO·비GMO 식품에 대한 학급·교사·학부모 인식을 직접 설문 수집 비교 후 탐구보고서 작성
★ 식품 종류·설문 대상을 바꿔서 설계해도 좋아요
활동3
1학년 1학기: 식품 포장재 보존 효과 비교 실험
예시: 랩·지퍼백·유리병·알루미늄 호일의 식품 보존 효과를 색·질감·무게 변화로 직접 측정 비교
★ 포장재 종류·식품 유형을 바꿔서 설계해도 좋아요
1학년 2학기: 전통·현대 발효 기술 비교 실험
예시: 전통 방식(항아리)과 현대 방식(냉장·밀봉)으로 같은 식품을 발효시켜 맛·pH·색 변화를 직접 측정 비교
★ 발효 식품 종류·보관 조건을 바꿔서 설계해도 좋아요
2학년 1학기: 식품 위생 실태 비교 실험
예시: 조리 전후·보관 방법별 식품 표면 오염도를 형광 로션·ATP 측정기로 직접 비교 관찰·기록
★ 조리 방법·보관 조건을 바꿔서 설계해도 좋아요
2학년 2학기: 미래 식량 인식 설문 조사
예시: 곤충·배양육·식물성 고기 등 미래 식량에 대한 학급·교사·학부모 인식을 직접 설문 수집해 세대별 비교
★ 미래 식량 유형·설문 대상을 바꿔서 설계해도 좋아요
3학년 1학기: 대체 식품 소비자 인식 탐구보고서
예시: 배양육·식물성 고기·곤충 단백질 시식 후 학급 내 반응·구매 의향을 직접 수집 비교 후 탐구보고서 작성
★ 대체 식품 종류·측정 항목을 바꿔서 설계해도 좋아요
활동4
1학년 1학기: 식품 알레르기 인식 설문 조사
예시: 학급·교사·학부모 대상 식품 알레르기 경험·인식·대처 방법을 직접 설문 수집해 비교 분석
★ 알레르기 유형·설문 대상을 바꿔서 설계해도 좋아요
1학년 2학기: 발효식품과 장 건강 인식 조사
예시: 김치·요거트·된장 등 발효식품 섭취 습관과 소화·건강 상태를 학급 내 직접 설문 수집 상관관계 분석
★ 발효식품 종류·설문 항목을 바꿔서 설계해도 좋아요
2학년 1학기: 식품 보존 방법 비교 실험
예시: 냉장·냉동·건조·염장·진공 방식으로 같은 식품을 보관해 기간별 색·질감·무게 변화를 직접 측정 비교
★ 보존 방법·식품 종류를 바꿔서 설계해도 좋아요
2학년 2학기: 스마트팜 인식 설문 조사
예시: 식물 공장·수직농업·스마트팜에 대한 학급·교사·학부모·농업 종사자 인식을 직접 설문 수집해 비교 분석
★ 농업 기술 유형·설문 대상을 바꿔서 설계해도 좋아요
3학년 1학기: 곤충 단백질 지속 가능성 탐구보고서
예시: 곤충 단백질 식품을 직접 시식해 학급 내 반응·수용도 측정 + 환경적 이점 인식 설문 수집 후 탐구보고서 작성
★ 곤충 단백질 종류·측정 항목을 바꿔서 설계해도 좋아요

계열별 탐구활동 로드맵 (5단계)
형식: 활동명 / 예시 / ★(변형 아이디어)
단계: 1학년1학기 → 1학년2학기 → 2학년1학기 → 2학년2학기 → 3학년1학기
계열: 경영/경제

💰 경영/경제
활동1
1-1: 경제 뉴스 소비 패턴 설문 / 학급·교사·학부모 대상 경제 뉴스 소비 방식·신뢰도 설문 수집해 세대별 비교 / ★설문 대상·문항 변경 가능
1-2: 창업 아이디어 소비자 반응 조사 / 가상 창업 아이디어 직접 설계하고 학급 내 소비자 반응·구매 의향 설문 수집 / ★아이디어 종류·설문 대상 변경 가능
2-1: 소비자 구매 패턴 관찰 / 학교 매점·편의점에서 시간대별·요일별 구매 패턴 직접 관찰·기록해 행동경제학 이론과 연결 / ★관찰 장소·시간대 변경 가능
2-2: 글로벌 시장 진출 전략 설문 / 해외 시장 진출 관심 제품 선정하고 수요 설문 수집 분석 / ★타깃 시장·제품 종류 변경 가능
3-1: 스타트업 IR 피칭 탐구보고서 / 창업 아이디어의 시장 수요·경쟁사·소비자 반응 데이터 수집 후 IR 자료 작성 / ★아이디어 분야·수집 방법 변경 가능
활동2
1-1: 지역 경제 현장 조사 / 학교 주변 소상공인 5인 이상 직접 인터뷰해 경기 체감 지수·매출 변화 수집 / ★업종·지역 변경 가능
1-2: 브랜드 인지도 설문 / 동일 카테고리 브랜드 학급 내 인지도·선호도 직접 설문 수집 비교 / ★브랜드 카테고리·설문 대상 변경 가능
2-1: 기업 재무 데이터 비교 / 동일 업종 기업 2~3개의 공개 재무제표 직접 수집해 매출·영업이익·부채비율 비교 / ★업종·비교 기간 변경 가능
2-2: 환율 변동과 소비자 반응 / 환율 변동 시기별 수입품 가격 변화 수집하고 소비 행태 변화 설문 분석 / ★수입품 종류·환율 변동 시기 변경 가능
3-1: CSR 경영 탐구보고서 / 기업 CSR 활동에 대한 소비자 인식·구매 의향 변화 직접 설문 수집 분석 / ★기업 종류·CSR 활동 유형 변경 가능
활동3
1-1: 경제 데이터 시각화 / 학급 내 용돈·소비 패턴 데이터 직접 수집해 엑셀·구글 시트로 시각화 분석 / ★수집 변인·시각화 방식 변경 가능
1-2: 사회적 기업 인식 설문 / 사회적 기업 제품 구매 의향·인식을 학급·지역 주민 대상 직접 설문 수집 비교 / ★설문 대상·기업 유형 변경 가능
2-1: 금융 상품 비교 분석 / 은행·증권사의 예금·적금·펀드 상품 조건 직접 수집해 수익률·리스크 비교 / ★금융 상품 종류·비교 기간 변경 가능
2-2: 국제 경제 협약 모의 협상 / 실제 무역 분쟁 사례 선정해 이해관계자 역할 나눠 모의 협상 진행 후 과정·결과 기록 / ★협상 주제·역할 배분 변경 가능
3-1: 경영 사례 탐구보고서 / 성공·실패 기업 사례 선정 후 소비자 인식 설문 수집 + 원인 분석 / ★기업 종류·분석 관점 변경 가능
활동4
1-1: 경제 정책 인식 모의 토론 / 최저임금·기본소득 등 경제 정책 학급 내 찬반 설문 수집 후 데이터 기반 모의 토론 / ★토론 주제·설문 문항 변경 가능
1-2: 광고 전략 효과 비교 / 동일 제품의 감성·이성·공포 소구 광고 직접 제작해 학급 내 반응·구매 의향 비교 / ★광고 유형·제품 종류 변경 가능
2-1: 소비자 가격 민감도 / 동일 제품의 가격을 다르게 제시해 학급 내 구매 의향 변화 직접 측정(앵커링 효과 검증) / ★제품 종류·가격 차이 변경 가능
2-2: 환율 변동 분석 프로젝트 / 최근 1년간 주요 통화 환율 데이터 직접 수집해 변동 원인·패턴 분석 후 소비자 영향 설문 / ★통화 종류·분석 기간 변경 가능
3-1: 경제 트렌드 탐구보고서 / 최근 소비 트렌드를 직접 설문·관찰로 데이터 수집 후 탐구보고서 작성 / ★트렌드 주제·수집 방법 변경 가능

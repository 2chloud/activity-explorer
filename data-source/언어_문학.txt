계열별 탐구활동 로드맵 (5단계)
형식: 활동명 / 예시 / ★(변형 아이디어)
단계: 1학년1학기 → 1학년2학기 → 2학년1학기 → 2학년2학기 → 3학년1학기
계열: 언어/문학

언어/문학 최종 정리본
활동1
1학년 1학기: 세대별 언어 사용 차이 조사
예시: 조부모·부모·본인 세대의 자주 쓰는 표현·사라진 말·새로 생긴 말을 직접 인터뷰·설문으로 수집 비교
★ 비교 세대·언어 유형을 바꿔서 설계해도 좋아요
1학년 2학기: 독자 반응 사전·사후 설문 조사
예시: 같은 소설·희곡 작품을 읽기 전후 감상 변화를 학급 내 직접 설문 수집해 비교 분석
★ 작품 종류·설문 문항을 바꿔서 설계해도 좋아요
2학년 1학기: 창작 영감 원천 인터뷰 조사
예시: 글쓰기·그림·음악 등 창작 활동을 하는 학급 내 학생·교사 대상 영감 원천을 직접 인터뷰 수집 비교
★ 창작 유형·인터뷰 대상을 바꿔서 설계해도 좋아요
2학년 2학기: 광고 언어 설득 효과 실험
예시: 동일 제품의 감성·이성·유머 소구 광고 카피를 직접 제작해 학급 내 설득력·구매 의향 비교 측정
★ 광고 유형·제품 종류를 바꿔서 설계해도 좋아요
3학년 1학기: 번역 품질 비교 탐구보고서
예시: 같은 외국 문학 작품의 번역본 2~3개를 직접 비교하고 모국어 화자 독자 반응 설문 수집 후 탐구보고서 작성
★ 작품 종류·번역본 수를 바꿔서 설계해도 좋아요
활동2
1학년 1학기: 문학 장르별 독자 선호도 설문 조사
예시: 학급·교사·학부모 대상 소설·시·수필·희곡 선호도와 이유를 직접 설문 수집해 세대별 비교
★ 장르 종류·설문 대상을 바꿔서 설계해도 좋아요
1학년 2학기: 한국·세계 문학 감상 비교 조사
예시: 같은 주제(사랑·죽음·성장 등)의 한국·외국 문학 작품을 읽고 학급 내 감상 차이를 직접 설문 수집 비교
★ 주제·비교 작품을 바꿔서 설계해도 좋아요
2학년 1학기: 드라마·웹소설 소비 패턴 조사
예시: 학급·학부모 대상 드라마·웹소설 소비 방식·선호 서사 구조를 직접 설문 수집해 세대별 비교
★ 콘텐츠 유형·설문 대상을 바꿔서 설계해도 좋아요
2학년 2학기: 언어의 감성적 효과 실험
예시: 같은 내용을 긍정·부정·중립 표현으로 달리 작성해 학급 내 감정 반응·설득력을 직접 측정 비교
★ 표현 방식·내용 유형을 바꿔서 설계해도 좋아요
3학년 1학기: 미디어 속 은유·상징 분석 탐구보고서
예시: 최근 광고·영화·드라마에서 사용된 은유·상징을 직접 수집 분석하고 학급 내 인식 설문 수집 후 탐구보고서 작성
★ 미디어 종류·분석 기간을 바꿔서 설계해도 좋아요
활동3
1학년 1학기: 문자 체계별 학습 난이도 비교 조사
예시: 한글·로마자·한자·아랍 문자를 처음 배우는 외국인·다문화 학생 대상 학습 경험을 직접 인터뷰 수집 비교
★ 문자 종류·인터뷰 대상을 바꿔서 설계해도 좋아요
1학년 2학기: 대중문학·순수문학 독자 인식 조사
예시: 웹소설·베스트셀러·순수문학 독자 대상 읽는 이유·만족도를 직접 설문 수집해 비교 분석
★ 문학 유형·설문 대상을 바꿔서 설계해도 좋아요
2학년 1학기: 플롯 구조와 독자 몰입도 실험
예시: 결말 공개·비공개·반전 구조로 같은 이야기를 달리 제시해 학급 내 몰입도·흥미도 비교 측정
★ 플롯 유형·측정 방법을 바꿔서 설계해도 좋아요
2학년 2학기: SNS 언어 변화 실태 조사
예시: 학급 내 SNS 사용 언어(줄임말·이모티콘·신조어)를 직접 수집하고 교사·학부모와 비교 분석
★ SNS 플랫폼·언어 유형을 바꿔서 설계해도 좋아요
3학년 1학기: 미래 언어 변화 예측 탐구보고서
예시: 세대별 언어 변화 인식을 직접 설문 수집하고 최근 10년간 신조어 데이터 분석 후 탐구보고서 작성
★ 언어 변화 유형·설문 대상을 바꿔서 설계해도 좋아요
활동4
1학년 1학기: 번역 품질 독자 반응 비교 실험
예시: 같은 외국 문학 작품의 번역본 2~3개를 학급 내 독자에게 제시해 자연스러움·충실도 직접 비교 측정
★ 작품 종류·평가 기준을 바꿔서 설계해도 좋아요
1학년 2학기: 역사적 사건과 문학 인식 조사
예시: 역사적 사건(독립운동·전쟁·민주화 등)을 다룬 문학 작품을 읽고 학급 내 감상·역사 인식 변화를 직접 설문 수집
★ 역사 사건·작품 종류를 바꿔서 설계해도 좋아요
2학년 1학기: 인터랙티브 스토리텔링 반응 실험
예시: 독자가 결말을 선택하는 인터랙티브 이야기를 직접 제작해 학급 내 몰입도·만족도 비교 측정
★ 이야기 주제·선택지 수를 바꿔서 설계해도 좋아요
2학년 2학기: 영화·문학 매체 변환 반응 조사
예시: 같은 작품의 소설·영화·드라마 버전을 경험한 학급 내 독자·관객 반응을 직접 설문 수집 비교
★ 작품 종류·매체 유형을 바꿔서 설계해도 좋아요
3학년 1학기: 디지털 글쓰기 변화 탐구보고서
예시: 손글씨·워드·SNS·AI 글쓰기 방식별 표현 특성과 독자 반응을 직접 수집 분석 후 탐구보고서 작성
★ 글쓰기 방식·분석 기준을 바꿔서 설계해도 좋아요

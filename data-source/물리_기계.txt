계열별 탐구활동 로드맵 (5단계)
형식: 활동명 / 예시 / ★(변형 아이디어)
단계: 1학년1학기 → 1학년2학기 → 2학년1학기 → 2학년2학기 → 3학년1학기
계열: 물리/기계

⚙ 물리/기계
활동1
1-1: 스포츠 속 물리 법칙 측정 / 농구공·테니스공·탁구공의 낙하 높이별 반발 계수 직접 측정 / ★공 종류·바닥 재질 변경 가능
1-2: 기계적 에너지 보존 / 경사면 각도별 구슬의 속도·높이 변화를 측정해 역학적 에너지 보존 검증 / ★경사면 재질·구슬 질량 변경 가능
2-1: 전자기파 차폐 효과 측정 / 알루미늄·구리·종이·천 등 재료별 전자기파 차폐 효과를 스마트폰 신호 강도로 비교 / ★재료 두께·겹침 수 변경 가능
2-2: 교류·직류 특성 비교 / 건전지(직류)·어댑터(교류) 전원의 전압·전류 특성을 멀티미터로 직접 측정 비교 / ★전압 크기·부하 저항 변경 가능
3-1: 자기장 가둠 탐구보고서 / 자석 배열에 따른 자기장 세기·분포를 나침반·철가루로 직접 측정 비교 / ★자석 개수·배열 방식 변경 가능
활동2
1-1: 힘의 분산 구조 / 삼각형·사각형·아치형 종이 구조물의 하중 지지력 직접 측정 비교 / ★재료 두께·구조물 크기 변경 가능
1-2: 탄성 에너지와 스포츠 장비 / 고무줄·용수철·탄성밴드의 탄성계수 직접 측정 + 발사체 속도와 연결 / ★재료 종류·길이·두께 변경 가능
2-1: 베르누이 법칙 / 종이·탁구공을 이용해 유속 변화에 따른 압력 차이 직접 관찰·측정 / ★관 직경·유속 변경 가능
2-2: 전자기 유도 / 코일 감는 수·자석 속도 달리해 유도 전류 크기 직접 측정 비교(패러데이 법칙 검증) / ★코일 재질·자석 세기 변경 가능
3-1: 미래 에너지 탐구보고서 / 태양광·풍력·수력 소형 모델의 발전 효율 직접 측정 비교 / ★발전 조건 변경 가능
활동3
1-1: 드론 비행 원리 측정 / 프로펠러 크기·회전 속도별 양력 직접 측정 비교(간이 장치 제작) / ★프로펠러 날 수·각도 변경 가능
1-2: 자기력과 로렌츠 힘 / 전류 세기·자석 배치 달리해 도선에 작용하는 힘의 크기 직접 측정 비교 / ★전류 방향·자석 세기 변경 가능
2-1: 로켓 추진 원리 / 물로켓 물 충전량·발사 각도별 비행 거리·높이 직접 측정 비교 / ★로켓 무게·날개 크기 변경 가능
2-2: 열팽창 측정 / 구리·철·알루미늄·유리의 온도별 길이 변화 직접 측정 비교 / ★재료 길이·온도 범위 변경 가능
3-1: 고속철도 마찰열 탐구보고서 / 재료별 마찰 계수와 마찰열 발생량 직접 측정 비교 / ★표면 거칠기·하중 변경 가능
활동4
1-1: 탄성 충돌과 비탄성 충돌 / 쇠구슬·고무공·찰흙공의 충돌 전후 속도 직접 측정해 운동량 보존 검증 / ★구슬 질량·충돌 각도 변경 가능
1-2: 에너지 효율 비교 / 소형 전동 모터카와 고무줄 동력 모형차의 에너지 투입 대비 이동 거리 비교 / ★모터 전압·고무줄 탄성 변경 가능
2-1: 냉매 기술 원리 / 에탄올·아세톤·물의 증발 냉각 효과를 피부 온도계로 직접 측정 비교 / ★증발 면적·바람 세기 변경 가능
2-2: 단열재 효과 비교 / 스티로폼·신문지·공기층·알루미늄 호일의 단열 효과를 온도 변화로 직접 측정 / ★단열재 두께·겹침 수 변경 가능
3-1: 광학 원리 탐구보고서 / 볼록렌즈·오목렌즈·프리즘의 빛 굴절·초점 거리 직접 측정 비교 / ★렌즈 곡률·빛의 파장 변경 가능

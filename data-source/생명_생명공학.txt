계열별 탐구활동 로드맵 (5단계)
형식: 활동명 / 예시 / ★(변형 아이디어)
단계: 1학년1학기 → 1학년2학기 → 2학년1학기 → 2학년2학기 → 3학년1학기
계열: 생명/생명공학

🧬 생명/생명공학
활동1
1-1: 세포 구조 관찰 실험 / 양파 표피·구강 상피 세포 직접 채취 후 현미경 관찰·스케치 비교 / ★재료 변경 가능
1-2: 자극과 반응속도 측정 실험 / 시각·청각 자극별 반응속도 학급 내 20명 이상 직접 측정 비교 / ★자극 종류 변경 가능
2-1: 운동과 신체 변화 측정 실험 / 운동 강도별 심박수·체온 자가 측정 후 집단 데이터 수집 / ★측정 변인 변경 가능
2-2: 천연 항균 물질 비교 실험 / 마늘즙·녹차·식초·소금물을 화장솜에 적셔 빵 곰팡이 억제 효과 비교 / ★종류·농도 변경 가능
3-1: 대체 단백질 비교 탐구보고서 / 배양육·곤충·식물성 단백질 성분표 비교 + 소비자 인식 설문 수집 / ★비교 대상 변경 가능
활동2
1-1: 효소 활성 비교 실험 / 침 속 아밀라아제 온도별 활성 변화를 아이오딘 반응으로 측정 / ★pH·기질 농도 변경 가능
1-2: 식물 효소 비교 실험 / 파인애플·키위·생강의 단백질 분해 능력을 젤라틴 굳기로 비교 / ★가열 여부·농도 변경 가능
2-1: 수면·카페인과 집중도 상관관계 조사 / 학급 내 수면 시간·카페인 섭취량과 집중도 설문 수집 후 분석 / ★변인 변경 가능
2-2: 천연 항균 물질 비교 실험 / 마늘·생강·녹차·자몽씨 추출물의 항균 효과를 빵 곰팡이 억제로 비교 / ★추출 방법·농도 변경 가능
3-1: 생분해 소재 탐구보고서 / 우무·전분 기반 바이오플라스틱 직접 제작 후 강도·분해 속도 비교 / ★소재·첨가물 변경 가능
활동3
1-1: 광합성 영향 요인 실험 / 빛의 색(파장)에 따른 엘로디아 기포 발생 속도 비교 / ★빛 세기·CO₂ 농도 변경 가능
1-2: 근육 피로 측정 실험 / 운동 강도별 근육 피로도·회복 시간 자가 측정 후 개인별 차이 비교 / ★측정 부위·운동 종류 변경 가능
2-1: DNA 추출 및 비교 실험 / 딸기·바나나·양파 DNA 추출 후 추출량·점성·색 비교 / ★추출 조건 변경 가능
2-2: 수질 오염 및 정화 실험 / 수돗물·정수기물·빗물 pH·탁도 직접 측정 비교 / ★정화 방법 변경 가능
3-1: 바이오플라스틱 탐구보고서 / 우무·전분·한천 소재별 바이오플라스틱 제작 + 물리적 특성 비교 / ★첨가물 종류·비율 변경 가능
활동4
1-1: 식물 DNA 추출 비교 실험 / 딸기·바나나·키위 DNA 추출 후 추출량·점성 비교 / ★추출 조건 변경 가능
1-2: 카페인과 신체 반응 측정 실험 / 카페인 음료 섭취 전후 심박수·반응속도 자가 측정 비교 / ★카페인 양·음료 종류 변경 가능
2-1: GMO 식품 인식 조사 / 시중 GMO·비GMO 성분표 비교 + 학급 내 GMO 인식 설문 수집 / ★비교 식품군 변경 가능
2-2: 식물 생장 환경 변인 실험 / 온도·빛·수분 조건 달리해 교내 식물 생장 속도 비교 / ★식물 종류·변인 조합 변경 가능
3-1: 감미료 역치 탐구보고서 / 천연·인공 감미료 농도별 단맛 인식 역치를 학급 내 실험으로 측정 / ★감미료 종류·피험자 그룹 변경 가능

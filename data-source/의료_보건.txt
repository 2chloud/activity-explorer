계열별 탐구활동 로드맵 (5단계)
형식: 활동명 / 예시 / ★(변형 아이디어)
단계: 1학년1학기 → 1학년2학기 → 2학년1학기 → 2학년2학기 → 3학년1학기
계열: 의료/보건

의료/보건 최종 정리본
활동1
1학년 1학기: 운동과 심혈관계 변화 측정 실험
예시: 운동 강도별(걷기·달리기·줄넘기) 심박수·혈압 변화를 학급 내 직접 측정해 비교 분석
★ 운동 종류·측정 시간을 바꿔서 설계해도 좋아요
1학년 2학기: 공기질과 호흡기 반응 조사
예시: 교내 교실·복도·운동장의 CO₂ 농도·미세먼지를 시간대별로 직접 측정 비교
★ 측정 장소·시간대를 바꿔서 설계해도 좋아요
2학년 1학기: 손 위생과 감염 예방 실험
예시: 손 씻기 방법별(물만·비누·소독제) 형광 로션으로 잔여 오염 정도를 직접 비교 관찰·기록
★ 세척 시간·방법을 바꿔서 설계해도 좋아요
2학년 2학기: 백신·면역 인식 설문 조사
예시: 학급·학부모·지역 주민 대상 백신 접종 경험·인식·신뢰도를 직접 설문 수집해 세대별 비교
★ 백신 종류·설문 대상을 바꿔서 설계해도 좋아요
3학년 1학기: 의료 AI 인식 탐구보고서
예시: 의료 AI 활용에 대한 환자·의료진·일반인 인식 차이를 직접 설문 수집 분석 후 탐구보고서 작성
★ 설문 대상·AI 활용 분야를 바꿔서 설계해도 좋아요
활동2
1학년 1학기: 뇌와 집중력 측정 실험
예시: 카페인·수면·운동 조건별 집중력(반응속도·기억력 테스트)을 학급 내 직접 측정 비교
★ 측정 조건·테스트 방식을 바꿔서 설계해도 좋아요
1학년 2학기: 영양소와 신체 변화 조사
예시: 학급 내 식습관·영양소 섭취량을 직접 설문 수집하고 신체 활력·집중도와 상관관계 분석
★ 영양소 종류·측정 변인을 바꿔서 설계해도 좋아요
2학년 1학기: 스트레스와 신체 반응 측정 실험
예시: 시험 전후·휴식 전후 심박수·혈압·피부 온도를 직접 측정해 스트레스 지표 비교
★ 스트레스 유발 조건·측정 변인을 바꿔서 설계해도 좋아요
2학년 2학기: 항생제 남용 인식 설문 조사
예시: 학급·학부모·지역 주민 대상 항생제 사용 경험·인식을 직접 설문 수집해 세대별 비교 분석
★ 설문 대상·문항을 바꿔서 설계해도 좋아요
3학년 1학기: 재생 의료 인식 탐구보고서
예시: 줄기세포·인공 장기 등 재생 의료에 대한 일반인·전문가 인식을 직접 설문 수집 후 탐구보고서 작성
★ 재생 의료 유형·설문 대상을 바꿔서 설계해도 좋아요
활동3
1학년 1학기: 면역과 생활 습관 상관관계 조사
예시: 학급 내 수면·운동·식습관과 감기 빈도를 직접 설문 수집해 상관관계 분석
★ 생활 습관 변인·설문 대상을 바꿔서 설계해도 좋아요
1학년 2학기: 유전과 가족 특성 조사
예시: 가족 내 혈압·시력·알레르기 등 유전적 특성을 직접 조사해 유전 패턴 분석
★ 조사 특성·가족 구성을 바꿔서 설계해도 좋아요
2학년 1학기: 공기 오염과 호흡기 건강 조사
예시: 교내외 미세먼지 농도를 시간대·장소별로 직접 측정하고 학급 내 호흡기 증상 설문과 비교
★ 측정 장소·시간대를 바꿔서 설계해도 좋아요
2학년 2학기: 개인 맞춤 의료 인식 설문 조사
예시: 유전자 검사·맞춤형 치료에 대한 학급·학부모·지역 주민 인식을 직접 설문 수집해 세대별 비교
★ 설문 대상·문항을 바꿔서 설계해도 좋아요
3학년 1학기: 줄기세포 치료 탐구보고서
예시: 줄기세포 치료 인식·윤리적 쟁점을 일반인·전문가 대상 직접 설문 수집 분석 후 탐구보고서 작성
★ 치료 유형·설문 대상을 바꿔서 설계해도 좋아요
활동4
1학년 1학기: 소화 효율 비교 실험
예시: 탄수화물·단백질·지방 식품의 소화 시간을 아이오딘·뷰렛 반응으로 직접 측정 비교
★ 식품 종류·소화 조건을 바꿔서 설계해도 좋아요
1학년 2학기: 감염병 확산 모의 실험
예시: 학급 내 모의 감염병 확산 시뮬레이션을 직접 설계·진행하고 마스크·거리두기 효과 비교 기록
★ 확산 조건·예방 방법을 바꿔서 설계해도 좋아요
2학년 1학기: 영양 불균형 실태 조사
예시: 학급 내 하루 식단을 직접 기록·수집해 영양소 섭취량 분석 후 불균형 패턴 도출
★ 식단 기록 기간·영양소 항목을 바꿔서 설계해도 좋아요
2학년 2학기: 감염병 공중보건 정책 인식 조사
예시: 코로나19 등 감염병 대응 정책에 대한 학급·학부모·지역 주민 인식을 직접 설문 수집 비교
★ 정책 종류·설문 대상을 바꿔서 설계해도 좋아요
3학년 1학기: 맞춤형 의학 탐구보고서
예시: 유전자 치료·정밀 의료에 대한 인식·윤리적 쟁점을 직접 설문 수집 분석 후 탐구보고서 작성
★ 의료 유형·설문 대상을 바꿔서 설계해도 좋아요

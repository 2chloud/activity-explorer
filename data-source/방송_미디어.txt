계열별 탐구활동 로드맵 (5단계)
형식: 활동명 / 예시 / ★(변형 아이디어)
단계: 1학년1학기 → 1학년2학기 → 2학년1학기 → 2학년2학기 → 3학년1학기
계열: 방송/미디어

활동1
1학년 1학기: 광고 스토리보드 제작 및 반응 실험
예시: 같은 제품의 감성·이성·유머 소구 광고 스토리보드를 직접 제작해 학급 내 설득력·구매 의향 비교 측정
★ 광고 유형·제품 종류를 바꿔서 설계해도 좋아요
1학년 2학기: 미니 다큐멘터리 제작 및 시청자 반응 조사
예시: 학교·지역 주제로 미니 다큐멘터리를 직접 기획·촬영하고 학급 내 시청자 반응·몰입도 설문 수집 비교
★ 주제·촬영 방식을 바꿔서 설계해도 좋아요
2학년 1학기: 단편 드라마 제작 및 시청자 반응 실험
예시: 같은 내용을 다른 장르(코미디·드라마·공포)로 직접 제작해 학급 내 시청자 반응·선호도 비교 측정
★ 장르·주제를 바꿔서 설계해도 좋아요
2학년 2학기: 프로그램 포맷 기획 및 반응 조사
예시: 새로운 방송 포맷을 직접 기획해 학급 내 시청 의향·관심도를 설문으로 수집 비교 분석
★ 포맷 유형·설문 대상을 바꿔서 설계해도 좋아요
3학년 1학기: 신규 프로그램 기획 탐구보고서
예시: 신규 프로그램을 직접 기획하고 학급·교사·학부모 대상 시청 의향·개선점 설문 수집 후 탐구보고서 작성
★ 프로그램 유형·설문 대상을 바꿔서 설계해도 좋아요
활동2
1학년 1학기: 개인 방송 채널 기획 및 반응 조사
예시: 유튜브·틱톡·인스타그램 채널을 직접 기획하고 학급 내 구독 의향·콘텐츠 선호도 설문 수집 비교
★ 플랫폼 종류·콘텐츠 유형을 바꿔서 설계해도 좋아요
1학년 2학기: 뉴스 문체 독자 반응 비교 실험
예시: 같은 사건을 공식·구어·SNS 문체로 각각 작성해 학급 내 신뢰도·이해도·선호도 비교 측정
★ 문체 유형·사건 종류를 바꿔서 설계해도 좋아요
2학년 1학기: 콘텐츠 마케팅 효과 비교 실험
예시: 같은 내용을 영상·카드뉴스·텍스트로 각각 제작해 학급 내 정보 전달력·공유 의향 비교 측정
★ 콘텐츠 형식·주제를 바꿔서 설계해도 좋아요
2학년 2학기: 미디어 산업 인식 세대별 비교 조사
예시: OTT·유튜브·TV 방송 소비 패턴을 학급·교사·학부모 대상 직접 설문 수집해 세대별 비교 분석
★ 미디어 유형·설문 대상을 바꿔서 설계해도 좋아요
3학년 1학기: 미디어 윤리 인식 탐구보고서
예시: 실제 미디어 윤리 위반 사례를 직접 수집하고 학급·교사·학부모 대상 인식 설문 후 탐구보고서 작성
★ 윤리 위반 유형·설문 대상을 바꿔서 설계해도 좋아요
활동3
1학년 1학기: 인터뷰 방식 효과 비교 실험
예시: 같은 주제로 공식·비공식·돌발 질문 방식의 인터뷰를 직접 진행해 응답 깊이·솔직함 비교 기록
★ 인터뷰 방식·주제를 바꿔서 설계해도 좋아요
1학년 2학기: 광고 음악 효과 비교 실험
예시: 같은 광고에 다른 배경음악(밝음·어두움·무음)을 적용해 학급 내 감정 반응·구매 의향 비교 측정
★ 음악 유형·광고 종류를 바꿔서 설계해도 좋아요
2학년 1학기: 미디어 소비 패턴 세대별 비교 조사
예시: 하루 미디어 소비 시간·플랫폼·콘텐츠 유형을 학급·교사·학부모 대상 직접 설문 수집해 세대별 비교
★ 미디어 유형·설문 대상을 바꿔서 설계해도 좋아요
2학년 2학기: 콘텐츠 트렌드 수집 분석
예시: 최근 3개월간 유튜브·틱톡·인스타그램 인기 콘텐츠를 직접 수집해 트렌드 패턴 분석 + 학급 내 인식 설문 비교
★ 플랫폼 종류·수집 기간을 바꿔서 설계해도 좋아요
3학년 1학기: 대본 분석 및 리라이트 탐구보고서
예시: 실제 방송 대본을 직접 수집·분석하고 시대·시청자 맞춤 리라이트 후 학급 내 반응 설문 수집 + 탐구보고서 작성
★ 대본 유형·리라이트 방향을 바꿔서 설계해도 좋아요
활동4
1학년 1학기: 방송 프로그램 시청자 반응 비교 조사
예시: 같은 장르의 한국·외국 방송 프로그램을 직접 시청하고 학급 내 선호도·차이점을 설문으로 수집 비교
★ 장르·비교 국가를 바꿔서 설계해도 좋아요
1학년 2학기: 포스터 디자인 효과 비교 실험
예시: 같은 프로그램의 포스터를 색상·구도·폰트를 달리해 직접 제작하고 학급 내 선호도·시선 집중 효과 비교 측정
★ 디자인 요소·프로그램 종류를 바꿔서 설계해도 좋아요
2학년 1학기: 미디어 사회적 영향 인식 설문 조사
예시: SNS·유튜브·TV가 학급 내 가치관·행동에 미치는 영향을 직접 설문 수집하고 교사·학부모와 비교 분석
★ 미디어 유형·영향 항목을 바꿔서 설계해도 좋아요
2학년 2학기: 브랜드 홍보 콘텐츠 효과 비교 실험
예시: 같은 브랜드의 홍보 콘텐츠를 영상·카드뉴스·텍스트로 직접 제작해 학급 내 브랜드 인지도·호감도 변화 비교 측정
★ 콘텐츠 형식·브랜드 종류를 바꿔서 설계해도 좋아요
3학년 1학기: 실시간 방송 기획 탐구보고서
예시: 실시간 방송을 직접 기획·진행하고 시청자 반응·채팅·참여율 데이터를 수집 분석 후 탐구보고서 작성
★ 방송 주제·플랫폼을 바꿔서 설계해도 좋아요

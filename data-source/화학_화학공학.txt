계열별 탐구활동 로드맵 (5단계)
형식: 활동명 / 예시 / ★(변형 아이디어)
단계: 1학년1학기 → 1학년2학기 → 2학년1학기 → 2학년2학기 → 3학년1학기
계열: 화학/화학공학

⚗ 화학/화학공학
활동1
1-1: 물질의 상태 변화와 에너지 측정 / 물·에탄올 가열 곡선 직접 측정, 상태 변화 구간 온도 변화 비교 / ★다른 순물질 변경 가능
1-2: 이산화탄소 흡수 효율 비교 / 식물·활성탄·소석회의 CO₂ 흡수 효율을 BTB 용액으로 비교 / ★흡수제 종류·양 변경 가능
2-1: 평형 이동 실험 / 온도·농도 변화에 따른 BTB 용액 색 변화 관찰·기록, 르샤틀리에 원리 검증 / ★산·염기 농도 변경 가능
2-2: 미세먼지 흡착 실험 / 활성탄·숯·커피찌꺼기의 색소(잉크) 흡착 효율 직접 측정 비교 / ★흡착제 입자 크기·시간 변경 가능
3-1: 신소재 탐구보고서 / 옥수수 전분·우무·천연고무 기반 소재 직접 제작 후 강도·유연성·열저항 비교 / ★소재·첨가물 변경 가능
활동2
1-1: 계면활성제 실험 / 비누·합성세제·주방세제의 기름 분리 효율과 거품 생성량 직접 비교 / ★농도·온도·경도 변경 가능
1-2: 플라스틱 물리적 특성 비교 / PET·PE·PP 플라스틱의 밀도·물 흡수율·열변형 온도 직접 측정 비교 / ★종류·측정 조건 변경 가능
2-1: 천연 방부제 효과 비교 / 식초·소금·설탕·꿀의 농도별 빵 곰팡이 억제 효과 육안으로 비교 / ★방부제 종류·농도 변경 가능
2-2: 생분해 플라스틱 제작 / 전분·우무·한천 기반 생분해 플라스틱 직접 제작 후 강도·투명도·분해 속도 비교 / ★첨가물 변경 가능
3-1: 탄소포집 탐구보고서 / 소석회·활성탄·해조류 분말의 CO₂ 흡수 효율을 BTB 용액으로 비교 측정 / ★포집 조건 변경 가능
활동3
1-1: 발열 반응 측정 / 손난로·베이킹파우더·염화칼슘의 발열량을 온도계로 직접 측정 비교 / ★반응물 양·온도 변경 가능
1-2: 휘발성 비교 / 에탄올·아세톤·식초·물의 증발 속도를 같은 조건에서 직접 측정 비교 / ★온도·표면적 변경 가능
2-1: 천연·합성 감미료 단맛 인식 / 설탕·스테비아·아스파탐·자일리톨 단맛 역치를 블라인드 테스트로 측정 / ★농도·피험자 변경 가능
2-2: 빛의 조건과 태양전지 효율 / 빛의 세기·각도·색에 따른 소형 태양전지 발전량 전류계로 직접 측정 / ★태양전지 종류·거리 변경 가능
3-1: 수소 생산 탐구보고서 / 전기분해 전압·전해질(소금·베이킹소다·식초) 농도별 기포 발생량 비교 / ★전극 재료 변경 가능
활동4
1-1: 산화 반응 속도 / 사과·감자·바나나의 갈변 속도를 조건별로 직접 측정 비교 / ★항산화 물질 종류·농도 변경 가능
1-2: 구강 제품 성분 비교 / 치약·구강청결제·베이킹소다의 pH·거품량·항균 효과 직접 측정 비교 / ★브랜드별·농도별 변경 가능
2-1: 의약품 성분 용해도 / 아세트아미노펜·아스피린·비타민C의 물·에탄올 용해도 차이 직접 측정 / ★온도·용매 종류 변경 가능
2-2: 연료 발열량 비교 / 양초·에탄올·식용유의 발열량을 물 가열 온도 변화로 직접 측정 비교 / ★연료 양·용기 크기 변경 가능
3-1: 인공광합성 탐구보고서 / 이산화티타늄 광촉매의 빛 조건별 색소 분해 속도 비교 실험 / ★광촉매 농도·색소 종류 변경 가능

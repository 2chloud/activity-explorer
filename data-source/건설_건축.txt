계열별 탐구활동 로드맵 (5단계)
형식: 활동명 / 예시 / ★(변형 아이디어)
단계: 1학년1학기 → 1학년2학기 → 2학년1학기 → 2학년2학기 → 3학년1학기
계열: 건설/건축

건설/건축 최종 정리본
활동1
1학년 1학기: 건축 재료 강도 비교 실험
예시: 종이·나무·플라스틱·빨대로 같은 구조물을 제작해 하중 지지력을 직접 측정 비교
★ 재료 종류·구조물 형태를 바꿔서 설계해도 좋아요
1학년 2학기: 고층 건물 구조 안전성 실험
예시: 다양한 높이·형태의 종이 구조물을 직접 제작해 진동·하중에 따른 내구성 비교 측정
★ 구조물 높이·재료를 바꿔서 설계해도 좋아요
2학년 1학기: 토양 하중 지지력 비교 실험
예시: 모래·점토·자갈·흙 등 토양 종류별 하중 지지력을 직접 측정 비교
★ 토양 종류·하중 크기를 바꿔서 설계해도 좋아요
2학년 2학기: 스마트시티 인식 설문 조사
예시: AI·IoT 기반 스마트시티 서비스에 대한 학급·교사·학부모·지역 주민 인식을 직접 설문 수집해 세대별 비교
★ 스마트시티 서비스 유형·설문 대상을 바꿔서 설계해도 좋아요
3학년 1학기: 건축 설계 탐구보고서
예시: 친환경·기능성·심미성을 고려한 소형 건축 모델을 직접 설계·제작하고 학급 내 평가 설문 수집 후 탐구보고서 작성
★ 설계 기준·모델 크기를 바꿔서 설계해도 좋아요
활동2
1학년 1학기: 트러스 구조 강도 비교 실험
예시: 삼각형·사각형·육각형 트러스 구조를 직접 제작해 하중 지지력·변형 정도 비교 측정
★ 구조 형태·재료를 바꿔서 설계해도 좋아요
1학년 2학기: 친환경 건축 소재 비교 실험
예시: 일반 시멘트·황토·재활용 소재의 강도·단열·흡수율을 직접 측정 비교
★ 소재 종류·측정 변인을 바꿔서 설계해도 좋아요
2학년 1학기: 도시 개발·환경 변화 현장 조사
예시: 학교 주변 개발 지역의 녹지·소음·공기질 변화를 직접 측정하고 주민 인터뷰로 인식 수집 비교
★ 조사 지역·환경 변인을 바꿔서 설계해도 좋아요
2학년 2학기: 전통·현대 건축 현장 비교 조사
예시: 지역 내 전통 건축물·현대 건축물을 직접 답사해 구조·재료·기능 차이를 측정·기록하고 방문객 인식 비교
★ 답사 장소·비교 항목을 바꿔서 설계해도 좋아요
3학년 1학기: 탄소 저감 건축 탐구보고서
예시: 소형 건축 모델에 단열재·태양광·자연환기를 적용해 에너지 효율을 직접 측정 비교 후 탐구보고서 작성
★ 적용 기술·측정 변인을 바꿔서 설계해도 좋아요
활동3
1학년 1학기: 단열 효과 비교 실험
예시: 스티로폼·신문지·에어캡·알루미늄 호일·나무의 단열 효과를 온도 변화로 직접 측정 비교
★ 단열재 종류·두께를 바꿔서 설계해도 좋아요
1학년 2학기: 교량 구조 강도 비교 실험
예시: 아치·현수·트러스·빔 교량 모형을 직접 제작해 하중 지지력·변형량 비교 측정
★ 교량 구조·재료를 바꿔서 설계해도 좋아요
2학년 1학기: 건축 디자인 선호도 비교 조사
예시: 같은 용도의 건물을 다른 디자인(미니멀·클래식·자연 친화)으로 직접 설계해 학급 내 선호도 설문 수집 비교
★ 디자인 유형·건물 용도를 바꿔서 설계해도 좋아요
2학년 2학기: 도심 재개발 인식 현장 조사
예시: 학교 주변 재개발 지역을 직접 답사해 현장 사진 수집하고 주민·상인 대상 인식 인터뷰 비교 분석
★ 답사 지역·인터뷰 항목을 바꿔서 설계해도 좋아요
3학년 1학기: 바이오 건축 소재 탐구보고서
예시: 대나무·볏짚·흙 등 바이오 기반 소재로 소형 구조물 직접 제작 후 강도·단열·지속 가능성 비교 + 탐구보고서 작성
★ 소재 종류·측정 변인을 바꿔서 설계해도 좋아요
활동4
1학년 1학기: 고대·현대 건축 구조 비교 현장 조사
예시: 지역 내 고대·현대 건축물을 직접 답사해 구조·재료·기능 차이를 측정·사진 수집하고 방문객 인식 비교
★ 답사 장소·비교 항목을 바꿔서 설계해도 좋아요
1학년 2학기: 유속과 구조물 내구성 실험
예시: 다양한 재료로 제작한 소형 구조물에 물 흐름을 직접 적용해 유속별 변형·내구성 비교 측정
★ 구조물 재료·유속 조건을 바꿔서 설계해도 좋아요
2학년 1학기: 도로·철도 내구성 설계 실험
예시: 다양한 재료(콘크리트·아스팔트 모형)의 하중·반복 충격에 따른 내구성을 직접 측정 비교
★ 재료 종류·하중 조건을 바꿔서 설계해도 좋아요
2학년 2학기: 건축 자재 방수 성능 비교 실험
예시: 콘크리트·방수 페인트·실리콘·타일의 방수 성능을 직접 물 흡수율·침투 속도로 측정 비교
★ 자재 종류·측정 조건을 바꿔서 설계해도 좋아요
3학년 1학기: 스마트 시티 탐구보고서
예시: 지역 내 스마트 시티 요소(교통·에너지·안전)를 직접 현장 조사하고 주민 인식 설문 수집 후 탐구보고서 작성
★ 스마트 시티 요소·조사 지역을 바꿔서 설계해도 좋아요

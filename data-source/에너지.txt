계열별 탐구활동 로드맵 (5단계)
형식: 활동명 / 예시 / ★(변형 아이디어)
단계: 1학년1학기 → 1학년2학기 → 2학년1학기 → 2학년2학기 → 3학년1학기
계열: 에너지

🔋 에너지
활동1
1-1: 에너지 보존 법칙 / 롤러코스터 모형 직접 제작해 높이별 구슬 속도 측정, 역학적 에너지 보존 검증 / ★경사면 각도·구슬 질량 변경 가능
1-2: 화석연료 연소 부산물 비교 / 양초·식용유·에탄올 연소 시 그을음·CO₂량을 석회수·BTB 용액으로 직접 비교 / ★연료 종류·연소 시간 변경 가능
2-1: 친환경 연료 발열량 비교 / 에탄올·식물성 기름·양초의 발열량을 물 가열 온도 변화로 직접 측정 비교 / ★연료 양·용기 크기 변경 가능
2-2: 탄소중립 인식 설문 / 학급·교사·학부모 대상 탄소중립 인식·실천 현황 설문 수집해 세대별 비교 / ★설문 대상·문항 변경 가능
3-1: 신재생에너지 전력 분배 탐구보고서 / 소형 태양광·풍력·수력 모델 조합해 조건별 전력 분배 효율 비교 / ★발전 조건·부하 종류 변경 가능
활동2
1-1: 열에너지 변환 / 구리·철·알루미늄·플라스틱의 열전도율 온도 변화로 직접 측정 비교 / ★재료 두께·길이 변경 가능
1-2: 발전소 유형별 효율 비교 / 소형 태양광·풍력·수력 발전 모델의 에너지 투입 대비 발전량 직접 측정 / ★발전 조건 변경 가능
2-1: 배터리 종류별 특성 비교 / 알칼리·니켈수소·리튬이온 건전지의 전압·방전 속도·온도 특성 직접 측정 / ★부하 저항·온도 조건 변경 가능
2-2: 배터리 방전 속도 측정 / 브랜드별·용량별 건전지의 방전 속도를 같은 부하 조건에서 직접 측정 비교 / ★부하 크기·온도 변경 가능
3-1: 탄소포집 탐구보고서 / 소석회·활성탄·해조류 분말의 CO₂ 흡수 효율을 BTB 용액으로 비교 측정 / ★포집 조건 변경 가능
활동3
1-1: 간이 발전기 제작 / 코일·자석으로 간이 발전기 직접 제작해 코일 감는 수·자석 세기별 발전량 비교 / ★코일 재질·자석 크기 변경 가능
1-2: 태양광 발전 효율 / 빛의 세기·각도·온도에 따른 소형 태양광 패널 발전량 직접 측정 비교 / ★패널 크기·광원 종류 변경 가능
2-1: 배터리 성능 비교 / 충전식·일회용 배터리의 용량·자가방전율·온도별 성능 직접 측정 비교 / ★충전 횟수·사용 환경 변경 가능
2-2: 에너지 빈곤 인식 설문 / 신재생에너지 보급 필요성·에너지 빈곤 인식을 학급·지역 주민 대상 설문 수집 / ★설문 대상·문항 변경 가능
3-1: 수소 에너지 탐구보고서 / 전기분해 전압·전해질 농도별 수소 발생량 직접 측정 비교 + 연료전지 원리 탐구 / ★전극 재료·전해질 종류 변경 가능
활동4
1-1: 연료 발열량 비교 / 양초·에탄올·식용유의 발열량을 물 가열 온도 변화로 직접 측정 비교 / ★연료 양·용기 크기 변경 가능
1-2: 풍력 발전기 블레이드 설계 / 블레이드 수·각도·재질별 풍력 발전량을 선풍기 바람으로 직접 측정 비교 / ★블레이드 길이·모양 변경 가능
2-1: 폐자원 에너지 생산 / 음식물 쓰레기·낙엽·종이의 퇴비화 발열량을 온도 변화로 직접 측정 비교 / ★폐자원 종류·분해 조건 변경 가능
2-2: 단열 효율 비교 / 스티로폼·신문지·공기층·알루미늄 호일의 단열 효과 온도 변화로 직접 측정 / ★단열재 두께·겹침 수 변경 가능
3-1: 에너지 자립형 건축물 탐구보고서 / 단열재·태양광·자연환기 조합한 소형 건축 모델 직접 제작 후 에너지 효율 비교 / ★건축 재료·설계 방식 변경 가능
